# AIアニメスタジオ ブランディング戦略リサーチレポート

**調査日**: 2026年2月24日
**調査方法**: Web検索による新規収集
**対象**: 国内アニメスタジオ18社 / 海外アニメーションスタジオ11社

---

## 1. エグゼクティブサマリー

本レポートは、国内外29社のアニメーション・アニメスタジオの事例を調査し、「受託メイン⇔オリジナル制作メイン」「人数規模（小⇔大）」の2軸でポジショニングマップを作成したものである。

**主要な発見:**
- **受託ゾーン（スコア0.0〜0.3）は競合密集**：国内の中堅〜大手スタジオの多くが受託制作を主力としており、価格・品質競争が激しい
- **オリジナル×小規模ゾーンに空白地帯**：少人数でオリジナルIP制作を行うスタジオは国内外ともに少数（京都アニメーション、Tonko House、Studio Colorido程度）
- **AI活用による差別化機会**：少人数体制でオリジナルIP制作を実現するには、AI技術による制作効率化が有効な差別化要素となりうる
- **成長パターンは3類型**：(1) 受託で規模拡大型、(2) 受託→オリジナル移行型、(3) 最初からオリジナル特化型

## 3. ポジショニングマップ解説

### 3.1 設立時マップ
設立時点では、ほぼ全てのスタジオが**少人数（2〜40人）**で創業している。例外はDreamWorks Animation（約250人、Amblimationのスタッフを吸収）とCloverWorks（A-1 Pictures高円寺スタジオからの分社化で約100人規模）。

設立時の受託/オリジナル傾向は現在と大きく異なるケースが多く、特に受託から出発してオリジナルに移行するパターンが顕著。

### 3.2 現在マップ
現在のマップでは以下の特徴が見られる:

**競合密集地帯:**
- **受託×中〜大規模（スコア0.1〜0.2、200〜400人）**: トムス、ufotable、CloverWorks、A-1 Pictures、ポリゴン・ピクチュアズ、オレンジ、DR Movie 等が集中
- **混合×中規模（スコア0.3〜0.5、200〜500人）**: MAPPA、WIT STUDIO、Production I.G、Fortiche、Haoliners 等

**空白地帯:**
- **オリジナル×小規模（スコア0.7〜1.0、20〜100人）**: Tonko House（20人）、Studio Colorido（40人）、サイエンスSARU（51人）のみ → **AI活用スタジオの参入余地が大きい**
- **オリジナル×大規模（スコア0.7〜1.0、500人以上）**: 海外メジャー（Pixar、DreamWorks、Illumination）のみが存在し、国内には不在

### 3.3 成長軌跡
主要スタジオの設立時→現在の成長軌跡から以下のパターンが読み取れる:
- **垂直成長型（受託のまま規模拡大）**: Titmouse（2→1,200人）、DR Movie（20→400人）
- **斜め成長型（規模拡大しつつオリジナル化）**: MAPPA、WIT STUDIO、ボンズ
- **オリジナル維持型（少人数のまま質を追求）**: Tonko House、Cartoon Saloon

## 6. 分析とインサイト

### 6.1 成長パターンの類型化

調査対象29社の成長パターンを以下の3類型に分類した:

#### Type A: 受託拡大型
規模を拡大しながら受託制作を主力として維持するパターン。

| スタジオ | 設立時→現在 | 特徴 |
|---------|------------|------|
| Titmouse | 2人→1,200人 | 独立系最大。受託一筋で急成長 |
| DR Movie | 20人→400人 | 韓国最大級の下請け。マッドハウス傘下 |
| ポリゴン・ピクチュアズ | 5人→300人 | 3DCG受託で海外展開 |
| トムス | 不明→307人 | 長寿IPの受託で安定成長 |

**特徴**: 安定した成長が見込めるが、IP保有による版権収入は限定的。価格競争に巻き込まれるリスクあり。

#### Type B: 受託→オリジナル移行型
受託制作で実力と規模を蓄えた後、自社オリジナルIPの制作・保有に移行するパターン。

**特徴**: 成長可能性が高いが、移行期に大きなリスクとリソース投入が必要。

#### Type C: オリジナル特化型

**特徴**: IP保有による長期的な収益が見込めるが、初期のキャッシュフローが厳しい。ヒット作が出るまでの資金繰りが課題。

### 6.2 空白地帯・競合密集地帯

#### 空白地帯（参入機会）

1. **オリジナル×小規模（スコア0.7-1.0、10-50人）**
   - 現在のプレイヤー: Tonko House（20人）、Colorido（40人）のみ
   - 参入障壁: 少人数でのオリジナルIP制作は制作効率が課題
   - **AI技術による制作効率化がこの空白地帯の参入を可能にする**

2. **オリジナル×大規模（スコア0.7以上、500人以上、国内）**
   - 国内にはこのポジションのスタジオが不在
   - 海外ではPixar/DreamWorks/Illuminationが占有

#### 競合密集地帯（参入注意）

1. **受託×中規模（スコア0.1-0.2、100-300人）**
   - 最も競合が多いゾーン。価格・品質競争が激しい
   - 新規参入は差別化が困難

2. **混合×中規模（スコア0.3-0.5、200-500人）**
   - MAPPA、WIT STUDIO等の急成長スタジオが競合

### 6.3 当社ポジショニング検討メモ

#### 提案ポジション: オリジナル×小規模（AI活用）
- **横軸スコア**: 0.75（オリジナルメイン）
- **縦軸**: 10-20人規模（創業時）
- **差別化要素**: AI技術による制作効率化

#### 近接競合とその差別化

| 競合 | ポジション | 当社との差別化 |
|------|-----------|-------------|
| Tonko House | オリジナル×超小規模 | 当社: AI技術、Tonko: Pixar OBの芸術性 |
| Studio Colorido | オリジナル×小規模 | 当社: AI技術、Colorido: ツインエンジングループの資本力 |
| サイエンスSARU | 混合×小規模 | 当社: AI技術・オリジナル特化、SARU: 監督の作家性 |

#### 参考にすべきモデル

1. **京都アニメーション型**: 自社レーベルによるIP内製化 → AI活用版として「AI＋クリエイターの共創による原作開発」を検討
2. **Fortiche型**: 特定IPホルダーとの深い協業 → ゲーム会社・Web小説プラットフォームとの連携
3. **Illumination型**: 本社（企画）+ 制作スタジオ（実制作）の分離 → AI活用で制作側のコスト効率化

## 7. 当社への示唆

### 7.1 ポジショニング戦略

**推奨**: 「AI技術を活用した少人数オリジナルIP制作スタジオ」としてのポジショニング

**理由:**
1. オリジナル×小規模ゾーンは明確な空白地帯であり、競合が少ない
2. AI技術は少人数でのオリジナルIP制作という従来不可能だった領域を可能にする
3. 受託ゾーンは競合密集しており、新規参入のメリットが薄い

### 7.2 成長シナリオ

**Phase 1（創業期）**: 10-15人規模、オリジナルIP 1-2作品の制作
- AI技術を活用した制作ワークフローの確立
- 短編/パイロット版での実績づくり

**Phase 2（成長期）**: 30-50人規模、オリジナルIP拡大 + 選択的受託
- ヒット作品によるブランド確立
- プラットフォーム（Netflix等）との直接契約
- 一部受託で収益安定化（サイエンスSARU型）

**Phase 3（確立期）**: 50-100人規模、IP事業の収益化
- 版権収入・マーチャンダイジングの拡大
- 京都アニメーション型のIP内製化モデルの確立

### 7.3 リスクと対策

| リスク | 対策 |
|--------|------|
| AI活用への業界・ファンの反発 | クリエイターとAIの共創を前面に出す。AI＝ツールとしての位置づけ |
| ヒット作品が出るまでの資金繰り | Phase 1で短編・パイロット中心にし投資を抑制。クラウドファンディングも検討 |
| AI技術の急速な陳腐化 | 技術そのものではなく「AI＋クリエイティブ」のワークフローで差別化 |
| 人材確保の困難 | リモート制作体制の構築。グローバルな人材ネットワーク |
